# 🛒 Zepto SQL Data Analysis

## 📌 Project Overview
This project analyzes Zepto's product dataset using PostgreSQL. The goal is to perform data cleaning, exploratory data analysis (EDA), and answer business-related questions using SQL queries.

## 🎯 Objectives
- Create a PostgreSQL database
- Import the Zepto dataset
- Perform data cleaning
- Analyze product and pricing data
- Generate business insights using SQL

## 🛠️ Technologies Used
- PostgreSQL
- pgAdmin 4
- SQL
- Git & GitHub

## 📂 Project Files
- `schema.sql` – Database schema
- `data.sql` – Data import script
- `queries.sql` – SQL analysis queries
- `README.md` – Project documentation

## 📊 SQL Concepts Used
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- HAVING
- Aggregate Functions
- CASE Statements
- JOINS
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions

## 📈 Business Insights
- Total number of products
- Category-wise product count
- Most expensive products
- Cheapest products
- Average product price
- Discount analysis
- Out-of-stock products
- Top-rated products

## 🚀 How to Run
1. Install PostgreSQL and pgAdmin.
2. Create a new database.
3. Import the dataset.
4. Run the SQL scripts.
5. Execute the analysis queries.
